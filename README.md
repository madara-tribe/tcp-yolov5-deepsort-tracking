# yolov5-deepsort-tracking via TCP Trasmission

# Env and Versions

## Host OS
- Mac OS 12.3.1
- python 3.6.10
- pytorch 1.10.2
- torchvision 0.11.3
- tkinter 8.6 

## guest OS
- ubuntu 20.04
- python 3.8.1
- pytorch 1.10.2+cu102
- torchvision 0.11.3+cu102


## How to Start

```
# at host OS
$ python3 tkapp.py
# at guest OS
$ python3 yolov5_detect.py
```
